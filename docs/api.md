# API Reference

Coming in Phase 5.

This document will cover all HTTP endpoints exposed by the Engine service, including request/response schemas, authentication, error formats, and usage examples.

## Planned Endpoints

- `POST /api/backtest/run` -- Submit a backtest run
- `GET /api/backtest/{id}` -- Retrieve backtest results
- `GET /api/signals/today` -- Get today's signal bar levels
- `POST /api/config` -- Create a strategy configuration
- `GET /api/health` -- Health check
